1. アプリの概要と基本操作

このアプリは、各工事の「予算・経費（材料費や外注費など）・請求・入金」を一元管理するためのアプリです。現場の担当者から経理担当者まで、それぞれの権限や業務に合わせてデータを入力・確認できます。

共通のボタン（アイコン）

➕
追加ボタン

新しいデータ（工事や経費など）を登録する時に押します。通常、画面の右下にあります。

✏️
編集ボタン

登録済みのデータを修正・変更する時に押します。

🗑️
削除ボタン

データを消去します。
※一度消すと元に戻せないので注意してください。

💾
保存 (Save)

入力画面で入力を終えたら、必ず右下の「Save」を押して保存してください。

2. 各種画面（メニュー）の見方

アプリを開くと、下部のタブやメニュー画面から目的の作業画面へ移動できます。

  • 📱 メニュー画面

    アプリのホーム画面です。ここから各機能にアクセスできます。

  • 一般・管理者
    🏢 工事管理画面 / 各種入力画面

    工事一覧の確認や、材料費・小口経費などの支払入力を行います。

  • 管理者・経理
    💰 経理管理画面 / 請求詳細画面

    月ごとの請求額、入金額、未入金の確認、支払いの管理を行います。

3. 新しい工事を登録する 一般・管理者 共通

新しい案件が始まったら、まずは「工事一覧」にデータを登録します。

1

メニューから 工事一覧（または工事管理画面）を開きます。

2

画面右下の ➕（追加）ボタン を押します。

3

フォームが開くので、必要な情報を入力します。

  • 得意先: ドロップダウンから選びます（事前に得意先マスタに登録が必要です）。
  • 現場名: 現場の名前を入力します。
  • 営業 / 管理: 担当する社員を選びます。
  • 予算: 「予算材料費」「予算手間代」「予算経費」を入力すると、自動で「予算支払総計」が計算されます。
  • 工期（ヶ月） / 初回計上年月: スケジュールを入力します。
4

右下の Save（保存） を押して登録完了です。

4. 支払入力（材料費・小口経費・カード決済） 一般・管理者 共通

工事にかかった経費は、種類に合わせて専用の画面から入力します。入力した金額は、自動的に対象の工事の「実績」として集計されます。

📦 材料費入力

仕入先から購入した材料費の入力です。

  1. 「材料費入力画面」を開く
  2. ➕ボタンを押す
  3. 「工事ID」「計上年月」「仕入先名」「金額」を入力
  4. Saveする

💴 小口経費精算

駐車場代、備品代、飲食代などの入力です。

  1. 「小口経費入力画面」を開く
  2. ➕ボタンを押す
  3. 「工事ID」「用途」「金額」を入力
  4. 必要に応じて「証憑」に写真を添付
  5. Saveする

💳 カード決済

会社用クレジットカード等で決済したものの入力です。

  1. 「カード決済入力画面」を開く
  2. ➕ボタンを押す
  3. 「工事ID」「計上年月」「金額」を入力
  4. Saveする

5. 入金入力と請求管理 管理者・経理向け

主に経理担当者や管理者が利用する機能です。「経理管理画面」や「請求詳細画面」で各工事の月ごとの状況を確認し、入金実績や請求ステータスを更新します。

⚠️ 未請求アラート（赤字表示）について

当月以前のデータで、請求が「未請求」のままであるデータは、画面上で赤字で目立つように表示されます。請求漏れを防ぐための機能です。

🏦 入金の実績を登録する

お客様から入金があった際に入力します。

  1. 「経理管理画面」などを開き、該当するデータをタップします。
  2. ✏️（編集）ボタンを押します。
  3. 実際の入金方法に合わせて、「現金」「手形」「相殺」「会費」の項目に金額を入力します。
  4. 保存すると、自動的に「入金合計」や「実入金進捗率」、「月収支」が計算・更新されます。
💡 注意点

すべての入金が完了したら、対象の工事のステータスを「完工」に変更してください。

📨 請求済みに変更する

請求書の送付が完了した際に入力します。

  1. 「請求詳細画面」を開きます。
  2. 該当する工事月のデータをタップして詳細を開きます。
  3. 「請求済にする」ボタンを押すとステータスが変わり、送付日時が自動記録されます。
  4. ※間違えて押した場合は、「未請求に戻す」ボタンで元に戻せます。

6. マスタ（基本データ）の管理とロック設定 管理者向け

ドロップダウンで選択する項目（取引先や担当者）のリストや、アプリ全体の権限設定は各マスタ画面で管理します。

👥 得意先マスタ

顧客名、入金日、締め日、入金条件を登録します。工事登録時の選択肢になります。

🏢 仕入先マスタ

材料などを購入する業者の名前を登録します。材料費入力時の選択肢になります。

🧑‍💼 社員マスタ

アプリを利用する社員の名前、メールアドレス、部署を登録します。ここで「権限（管理者 / 一般）」の設定も行います。

🔒 設定マスタ (編集ロック)

過去データの改ざんを防ぐための重要な設定です。

「編集ロック年月」を設定すると、一般権限のユーザーはその月以前のデータを編集・削除できなくなります。（管理者はいつでも編集可能です）
